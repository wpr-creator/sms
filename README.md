# Falcon Adventure Camp

Flat GitHub Pages package. Copy all files into the root of the `sms` repo.

Files:
- `index.html`: camp homepage
- `flight-school.html`: math game
- `field-lab.html`: science placeholder
- `expedition-corps.html`: social studies placeholder
- `style.css`: shared design
- `camp.js`: homepage motion
- `flight-school.js`: game engine
- `question-bank.js`: editable math question bank

To publish:

```bash
git add .
git commit -m "Update Falcon Adventure Camp flat site"
git push origin main
```
